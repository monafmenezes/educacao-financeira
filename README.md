# 📘 Educação Financeira para Iniciantes

Caderno temático criado no **NotebookLM** com foco em educação financeira para iniciantes. O objetivo é **aprender a organizar finanças pessoais** de forma prática, usando vídeos e PDFs confiáveis.

## 🎯 Objetivos

- Compreender conceitos básicos de educação financeira  
- Aprender a controlar gastos e criar orçamento pessoal  
- Entender a importância de poupança e reserva de emergência  
- Ter noções iniciais de investimentos
